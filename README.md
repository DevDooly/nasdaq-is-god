# 증시 요약 알림 텔레그램 봇

매일 지정된 시간에 주요 증시 지수와 정보를 요약하여 텔레그램으로 알림을 보내는 봇입니다.

## 주요 기능

### 필수 기능
- **주요 지수 조회**:
  - S&P 500 지수 (`^GSPC`)
  - 나스닥 종합 또는 QQQ ETF (`^IXIC` / `QQQ`)
  - VIX 변동성 지수 (`^VIX`)
- **환율 정보**:
  - 주요 통화 환율 (예: USD/KRW)
- **개별 종목 조회**:
  - 특정 주식 종목의 현재가, 등락률 등 조회 기능 (예: `/stock TSLA`)

### 추천 기능
- **사용자 맞춤 알림**:
  - 사용자가 특정 종목의 가격 변동(예: +5% 이상, -5% 이하)에 대한 알림을 설정할 수 있는 기능.
- **포트폴리오 요약**:
  - 사용자가 자신의 포트폴리오(보유 종목, 수량)를 등록하면, 일일/주간 수익률 요약을 제공하는 기능.
- **주요 경제 뉴스 피드**:
  - 주요 경제 뉴스 헤드라인을 정해진 시간마다 받아볼 수 있는 기능.
- **암호화폐 시세 조회**:
  - 주요 암호화폐(예: 비트코인, 이더리움)의 시세를 조회하는 기능.
- **차트 이미지 생성**:
  - 특정 종목의 간단한 시세 차트 이미지를 생성하여 보여주는 기능.
- **정기 리포트**:
  - 매일 아침/저녁 장 마감 시황을 요약하여 리포트 형식으로 전송.
