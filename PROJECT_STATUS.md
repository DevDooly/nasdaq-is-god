# Project Status: Nasdaq is God

## 프로젝트 현재 상태 (2026-02-11 기준)

실전 매매를 위한 **한국투자증권(KIS) Open API 연동**을 완료했습니다.

### 1. 주요 변경 사항
- **실제 브로커 구현**: KIS Open API를 이용한 접근 토큰 발급, 잔고 조회, 미국 주식 주문 로직 구현 (`core/kis_broker.py`)
- **브로커 스위칭 시스템**: 환경변수(`USE_REAL_BROKER`) 설정을 통해 Mock 모드와 실전 매매 모드를 자유롭게 전환 가능
- **인증 보안 강화**: KIS API의 OAuth2 프로세스를 통합하여 자동 토큰 갱신 기능 추가

### 2. 현재 아키텍처
- **Backend**: Python(FastAPI) - 인증, 매매, 분석, 실전 브로커 통합 완료
- **Brokers**: `MockBroker` (테스트용), `KISBroker` (실전용) 선택 가능
- **Analysis Engine**: `IndicatorService` (실시간 지표 분석)
- **Frontend**: Flutter (연동 기초 작업 완료)

### 3. 향후 계획
- **대시보드 UI 구현**: 앱에서 실시간 잔고와 포트폴리오를 시각적으로 확인
- **자동매매 전략 워커**: 특정 지표 조건 만족 시 KISBroker를 통해 자동 주문 실행
- **백테스팅 시스템**: 실제 KIS 데이터를 활용한 전략 시뮬레이션
- **실시간 체결 알림**: 주문 체결 시 텔레그램 및 앱 푸시 알림 연동
