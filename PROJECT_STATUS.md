# 📊 Project Status: Nasdaq is God

## 프로젝트 현재 상태 (2026-02-11 기준)

단순 주식 조회 시스템을 넘어 **실시간 시세 연동, AI 뉴스 분석, 잔고 기반 매매 엔진**을 갖춘 풀스택 자동매매 플랫폼의 핵심 기능을 완성했습니다.

### ✅ 최근 완료된 주요 기능
- **실시간 시세 연동 (WebSocket)**: 9000번 포트를 통해 10초마다 실시간 주가 정보를 모든 클라이언트에 브로드캐스팅합니다.
- **AI 뉴스 감성 분석 (Gemini AI)**: 특정 종목의 최신 뉴스를 Gemini 1.5 Flash 모델이 분석하여 0~100점의 투자 심리 점수를 도출합니다.
- **잔고 관리 및 수익률 트래킹**: $100,000의 가상 원금을 기준으로 잔고 내에서만 구매 가능한 매매 로직과 누적 ROI 계산 기능을 구현했습니다.
- **시스템 안정화 (Integrity)**: `./scripts/check_system.sh`를 통해 백엔드 문법, 서버 포트, 프론트엔드 설정을 자동 점검하는 프로세스를 구축했습니다.
- **로그인 및 세션 유지**: 새로고침이나 브라우저 뒤로가기 시에도 로그인이 유지되는 자동 로그인(Auto-Login) 기능을 완성했습니다.

### 🏗️ 현재 시스템 아키텍처
- **Backend**: Python(FastAPI) - 9000 포트 (API & WebSocket)
- **Frontend**: Flutter Web - 8080 포트 (Threaded Server)
- **Engine**: TradingWorker (백그라운드 자동 매매), IndicatorService (RSI/MACD), AIService (Gemini)
- **Database**: PostgreSQL (Docker 기반)
