# Project Status: Nasdaq is God

## 프로젝트 현재 상태 (2026-02-11 기준)

사용자 편의를 위한 **실시간 대시보드 UI 및 포트폴리오 시각화 기초 작업**을 완료했습니다.

### 1. 주요 변경 사항
- **모의 매매 엔진 고도화**: `MockBroker`가 이제 고정가($150)가 아닌 **실제 현재 주가**를 연동하여 체결되도록 개선되었습니다.
- **인증 안정화**: 시크릿 모드 및 모바일 브라우저 환경에서도 로그인 세션이 유지되도록 메모리 토큰 백업 로직을 추가했습니다.
- **포트 충돌 해결**: API 서버 포트를 9000번으로 확정하여 TIL 위키와의 충돌을 해결했습니다.

### 2. 현재 아키텍처
- **Backend**: Python(FastAPI) - 인증, 매매, 분석, 실전 브로커 통합 완료
- **Frontend**: Flutter - 로그인 및 대시보드 UI 기능 구현 완료
- **Brokers**: `MockBroker` 및 `KISBroker` 연동 완료
- **Database**: PostgreSQL (사용자 및 자산 정보 관리)

### 3. 향후 계획
- **지표 차트 시각화**: `fl_chart`를 연동하여 종목별 RSI, MACD 지표를 그래프로 표시
- **매매 인터페이스 상세화**: 앱 내에서 직접 수량을 입력하여 매수/매도 주문을 넣는 트레이딩 폼 개발
- **자동매매 스케줄러**: 백엔드에서 특정 조건 시 자동 주문을 실행하는 백그라운드 워커 구축
- **실시간 데이터 스트리밍**: WebSocket을 이용한 가격 실시간 업데이트 반영
